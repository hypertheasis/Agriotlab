# AGRIOT 🌱
Agricultural Decision Support System for Home Assistant

## Τι είναι
Το **Agriot** είναι ένα ελαφρύ, επεκτάσιμο integration για Home Assistant
που βοηθά στη λήψη **γεωργικών αποφάσεων** (π.χ. σπορά, σταθερότητα καιρού)
χρησιμοποιώντας **Open-Meteo API** και κανόνες αγρονομίας.

## Τι κάνει (v0.1)
- 📡 Λαμβάνει μετεωρολογικά δεδομένα από Open-Meteo
- 🌡 Υπολογίζει θερμοκρασιακούς δείκτες
- 🌱 Παράγει αισθητήρες σποράς (θερμόφιλα / ψυχρόφιλα)
- 📅 Υποστηρίζει αυτοματισμούς & ημερολόγιο
- 🧠 Έτοιμο για AI advisory (επόμενες εκδόσεις)
